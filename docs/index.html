Math Problem Visualization

This graphic is an interactive illustration of the following mathematical object:
N nodes numbered 1,2,... N are arranged around a circle. A pointer to each node is then distributed either randomly or in order among all the nodes forming tuples (for example (1,2) => Node 1 pointing to Node 2 depicted below as arrows).
Additionally we can introduce a shift factor so that instead of pointing to the inherited pointer the node points to (node + s) % N where s is the shift factor.
